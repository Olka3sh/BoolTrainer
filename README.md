Назначение модулей:
 -  main.py - Бэкенд сервер
Назначение: Обработка логических вычислений и API

Основные компоненты:

Flask Application - веб-сервер и маршрутизация

Truth Table Calculator - генерация таблиц истинности

Normal Forms Transformer - преобразование в КНФ/ДНФ

Expression Parser - парсинг и валидация выражений

Ключевые функции:

calculate_truth_table() - построение таблиц истинности

calculate_normal_forms() - преобразование выражений

Обработка ошибок и валидация входных данных

 - templates/index.html - Пользовательский интерфейс
Назначение: Визуальное представление приложения

Компоненты:

Формы ввода - для выражений и переменных

Палитра блоков - логические элементы (AND, OR, NOT, XOR)

Рабочая область - для построения схем

Модальные окна - выбор действий

Области результатов - отображение таблиц и преобразований

 - static/js/app.js - Фронтенд логика
Назначение: Интерактивность и бизнес-логика клиента

Модули:

UI Controller - управление интерфейсом и событиями

Drag & Drop System - перетаскивание логических блоков

Connection Manager - система соединений между элементами

API Client - взаимодействие с бэкендом

Scheme Builder - построение и визуализация схем

 - static/css/style.css - Стилевая система
Назначение: Единый визуальный стиль и UX

Особенности:

CSS переменные для согласованной цветовой схемы

Стили для логических блоков и коннекторов

Адаптивный дизайн для разных размеров экранов

Анимации и переходы для улучшения пользовательского опыта
